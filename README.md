# calc

A simple command line calculator

## Documentation
Supports addition, subtraction, multiplication, division, and modulus.
Allows parentheses and ignores white space.

## Examples
```bash
node lib/calc.js '2+4*3'
#=> 14
```

```bash
node lib/calc.js '20 + ((4 - 7) / (2 * 10)) * 3'
#=> 19.55
```
